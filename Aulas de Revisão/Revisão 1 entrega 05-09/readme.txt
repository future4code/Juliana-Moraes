Loja no console

- Enunciado
    - Todo o projeto funciona através de interação por prompt e exibição no console.
    - Uma loja que possui 2 tipos de produtos: Alimentos e Brinquedos
        - Cada tipo de produto é um array que vai conter os itens disponíveis na loja (sempre 1 de cada tipo, no máximo)
    - Deve possuir as seguintes opções aceitas no prompt:
        - Vitrine: pergunta qual tipo de produto deseja ver
            - Deve aceitar as opções "Brinquedos" ou "Alimentos", e em seguida exibir no console a lista respectiva de cada um
        - Comprar: pergunta qual tipo de produto deseja ver
            - Deve aceitar as opções "Brinquedos" ou "Alimentos", e em seguida perguntar qual produto deseja comprar
            - Deve retirar esse elemento do array de itens respectivo e, em seguida, enviar uma mensagem de sucesso no console.
- Plano

    [Extensão para assistir vídeos em velocidade maior no Chrome.](https://chrome.google.com/webstore/detail/video-speed-controller/nffaoalbilbmmfgbnbgppjihopabppdk?hl=pt-BR)

    Siga o passo a passo a seguir, pulando as aulas caso se sinta confortável com o conteúdo.

    - Assistir aula Variáveis
    - Criar arrays contendo os itens da loja
    - Criar perguntas que serão usadas no projeto com prompt e armazenar em variáveis
    - Assistir aula Condicionais
    - Criar condicionais que decidirão o próximo passo em caso de
    - Assistir aula Laços
    - Criar laço para que sempre após realizar uma ação pedida, seja possível pedir outra.
    - Criar condição de saída para caso o usuário digite `sair`, o laço deve ser interrompido.
